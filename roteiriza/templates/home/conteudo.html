Bem vindo ao Roteiriza!

Como Funciona o Sistema de Roteirização

Objetos: Nesta página, é possível vizualizar e cadastrar os itens que serão entregues. Cada item recebe uma data de entrada, o que permite definir a prioridade de entrega. É possível registrar uma descrição para facilitar a identificação, telefone do destinatário para contato em caso de necessidade, endereço para a roteirização e uma indicação do tamanho do item (grande ou pequeno, sendo o último indicado para entregas via moto).

Além disso, é possível cadastrar múltiplos itens simultaneamente ao anexar uma planilha Excel na aba Incluir , utilizando o botão . Há também a opção Excluir tudo, que remove todos os itens cadastrados.

Veículos: Nesta seção, é vizualizado e cadastrado os veículos disponíveis para as entregas, incluindo a placa, apelido do veículo e a capacidade de transportar itens grandes. Como na seção de Objetos, há um botão Excluir tudo para remover todos os veículos registrados.

Rotas: Exibe as rotas geradas para cada veículo, com informações detalhadas e prontas para impressão em formato A4.

Configurações: Permite configurar a capacidade de transporte dos veículos (furgão e moto), o tempo máximo de expediente do motorista, o tempo médio estimado para cada entrega, e o endereço base de partida e retorno dos veículos.

Gerar Rotas: Este botão processa e atualiza as rotas conforme os dados cadastrados, garantindo que o sistema opere com informações atualizadas.

Sair: Utilizado para realizar o logout do sistema.
